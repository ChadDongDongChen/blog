Ubuntu Linux 安装完整教程

Ubuntu 是目前最流行的 Linux 发行版之一，以其友好的用户界面、强大的社区支持和稳定的系统性能而闻名。本教程将详细介绍 Ubuntu 的安装过程，无论是作为主系统、双系统还是在虚拟机中运行。

系统要求

在开始安装之前，请确保你的计算机满足以下最低要求：

  • 处理器：2 GHz 双核处理器或更高
  • 内存：4 GB RAM（推荐 8 GB 或以上）
  • 硬盘空间：25 GB 可用空间（推荐 50 GB 或以上）
  • 显示器：1024×768 分辨率
  • 启动介质：U盘（至少 4 GB）或 DVD
where

第一步：下载 Ubuntu 镜像

  1. 访问 Ubuntu 官网
  2. 选择适合的版本：
    • LTS 版本（长期支持版）：适合生产环境和普通用户，支持周期长达 5 年
    • 最新版本：包含最新的特性和软件，支持周期较短
  3. 点击下载按钮，获取 ISO 镜像文件（约 2-4 GB）

第二步：制作启动 U 盘

Windows 用户使用 Rufus

  1. 下载并安装 Rufus
  2. 插入 U 盘（注意：制作过程会格式化 U 盘，请备份重要数据）
  3. 打开 Rufus，选择：
    • 设备：你的 U 盘
    • 引导类型选择：选择下载的 Ubuntu ISO 文件
    • 分区类型：根据你的计算机选择 MBR（传统 BIOS）或 GPT（UEFI）
  4. 点击“开始”，等待制作完成

macOS 用户使用 Etcher

  1. 下载并安装 Etcher
  2. 插入 U 盘
  3. 打开 Etcher，选择 ISO 文件，选择 U 盘，点击“Flash!”
  4. 等待制作完成

第三步：启动安装程序

  1. 重启计算机，进入 BIOS/UEFI 设置（通常是按 F2、F10、F12 或 Delete 键）
  2. 设置 U 盘为第一启动项
  3. 保存设置并重启

第四步：安装 Ubuntu

1. 选择安装语言

启动后选择“Try Ubuntu”测试或“Install Ubuntu”直接安装

2. 选择键盘布局

默认选择“Chinese”即可

3. 连接网络（可选）

建议连接网络以下载更新和第三方软件

4. 选择安装类型

  • 正常安装：包含办公软件、媒体播放器等常用软件
  • 最小安装：仅包含基本系统，适合自定义安装
  • 其他选项：
    • 安装第三方软件：显卡驱动、媒体编码器等
    • 下载更新：在安装过程中下载最新更新

5. 分区设置（重要！）

方案一：清除整个磁盘安装（单系统）

  • 选择“清除整个磁盘并安装 Ubuntu”
  • 系统会自动创建所需分区

方案二：双系统安装

  • 选择“其他选项”手动分区
  • 推荐分区方案：
    • /（根分区）：30-50 GB，EXT4 格式
    • swap（交换分区）：内存大小的 1-2 倍（如果内存 ≥ 8 GB 可不设）
    • /home（家目录）：剩余空间，EXT4 格式（存储个人文件）

7. 创建用户账户

  • 输入你的姓名
  • 计算机名（会自动生成）
  • 用户名（用于登录）
  • 设置密码（建议设置强密码）

8. 开始安装

点击“安装”，等待安装完成（通常需要 20-40 分钟）

总结

Ubuntu 安装过程相对简单，但分区步骤需要谨慎操作。建议新手选择“清除整个磁盘安装”，而有经验的用户可以选择手动分区以更好地管理磁盘空间。

安装完成后，Ubuntu 提供了一个稳定、安全且免费的操作系统环境，无论是日常使用还是开发工作都能胜任。
